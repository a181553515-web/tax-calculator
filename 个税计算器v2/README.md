# 个人所得税计算器 v2.0-beta

一站式个税计算工具，覆盖预扣预缴、汇算清缴、经营所得等全场景。

## 功能模块

| 模块 | 支持场景 |
|------|---------|
| 预扣预缴 | 工资累计预扣、劳务报酬、稿酬、特许权使用费、非居民个人 |
| 汇算清缴 | 综合所得收入汇总，自动计算应补/应退税额 |
| 经营所得 | 查账征收、核定征收、分期预缴、个体工商户减半优惠 |
| 其他所得 | 财产租赁、财产转让、股息红利、偶然所得 |
| 特殊场景 | 年终奖两种计税对比、股权激励、离职补偿、个人养老金节税测算 |

## v2.0-beta 更新

- 全新视觉设计，移动端沉浸式体验
- 计算明细更详细，逐步骤展示公式与税率
- 税率速查表覆盖全部 8 类税目
- 分享图无长度限制，支持长图导出
- 输入示例占位符，上手更快

## 使用

浏览器打开 `index.html` 即用，无需安装。计算结果仅供参考，具体以税务机关认定为准。
