# 个人所得税计算器 v2.0-beta

一站式个税计算工具，覆盖预扣预缴、汇算清缴、经营所得等全场景。

## 功能模块

| 模块 | 支持场景 |
|------|---------|
| 预扣预缴 | 工资薪金（单月/累计估算/累计精算三模式）、保险证券律师、劳务报酬（一般/平台从业实习生累计预扣法）、稿酬、特许权使用费、非居民个人 |
| 汇算清缴 | 综合所得收入汇总（含保险佣金），专项/专项附加/其他扣除，自动计算应补/应退税额，表单可折叠，计算明细合并展示 |
| 经营所得 | 查账征收、核定征收、分期预缴、个体工商户减半优惠 |
| 其他所得 | 财产租赁、财产转让、股息红利（含持股期限差异）、偶然所得 |
| 特殊场景 | 年终奖两种计税对比、股权激励两种计税对比、离职补偿、个人养老金节税测算 |

## v2.0-beta 更新

- 工资薪金三模式：单月计算 / 累计估算（逐月明细表）/ 累计精算（含全年一次性减除开关）
- 新增保险证券律师特殊行业预扣模块
- 股权激励新增「并入综合所得」对比方式
- 劳务报酬新增「累计预扣法」模式，支持互联网平台从业人员和全日制学生实习
- 一键分享：结构化排版分享图，含卡片层级、计算明细、模块名称，支持年终奖/股权激励对比图
- 所有金额精确到分（2 位小数），输入自动清理前导零和多余小数位
- 汇算清缴表单默认折叠，计算明细合并为单一视图
- 仅输入收入字段时触发实时计算，扣除类字段不误触发
- 未输入数据时显示「请输入数据」，免税结果正确显示 ¥ 0.00
- 所有滑块默认关闭
- 视觉升级：Design Token 设计系统、ARIA 可访问性、三级响应式断点、字号/对比度 WCAG AA 合规

## 使用

浏览器打开 `index.html` 即用，无需安装。计算结果仅供参考，最终以税务机关认定为准。
